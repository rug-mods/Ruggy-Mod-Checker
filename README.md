Ruggy-Mod-Checker

Ruggy-Mod-Checker is a PC-only Gorilla Tag utility designed to assist in identifying potentially unsafe or unauthorized mods within a Gorilla Tag installation. The tool focuses on analyzing mod-related files to help users detect known problematic or suspicious modifications that may impact game stability, violate rules, or increase ban risk.

This project is intended exclusively for Gorilla Tag on PC and does not support Quest or standalone versions.

Key Characteristics

Targets Gorilla Tag mod environments on Windows PC

Performs automated checks instead of manual file inspection

Aims to improve mod safety, game stability, and user awareness

Lightweight and straightforward implementation

Purpose

The goal of Ruggy-Mod-Checker is to provide a simple, technical solution for players who want better visibility into what mods are present in their game directory, without relying on guesswork or manual scanning.

Built with practicality in mind for Gorilla Tag PC players who want to keep their setup clean and compliant.
